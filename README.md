# AI Resume Improvement Assistant (Prompt Engineering Project)

## Overview
This project uses OpenAI API and prompt engineering to improve resumes based on job descriptions.

## Features
- Resume analysis
- ATS optimization
- Keyword extraction
- Resume rewriting
- AI-powered feedback

## Technologies Used
- Python
- OpenAI API
- Prompt Engineering
- dotenv

## How to Run

```bash
pip install -r requirements.txt
python main.py
